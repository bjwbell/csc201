$\{i = N \wedge j = x[0] + x[1] + \ldots + x[N-1]\}$\\

$\{i + 1 = N \wedge j = x[0] + x[1] + \ldots + x[i+1 - 1]\}$\\
j := j + x[i];\\
$\{i + 1 = N \wedge j + x[i] = x[0] + x[1] + \ldots + x[i]\}$ \\
$\rightarrow \{i \le N \wedge j = x[0] + x[1] + \ldots x[i-1]\}$\\
